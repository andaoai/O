# 古籍笔记

本处按书立卷，按卷立章，按章立段。每一段落有独立编号（如 `yisizhan.01.001`），既方便自身回顾，也为将来罗盘与文本的**双向联动**留下锚点。

## 体例

每一段落分为四部分：

- **原文** —— 取自可靠底本，保留原始标点，用引用块呈现
- **校注** —— 版本差异、字词考据
- **我的注** —— 私人理解与联想，允许放胆推论
- **疑问** —— 未解之处，留待日后回读

段落 frontmatter 中会声明它讨论了哪些**罗盘环**（`rings:`）与**概念**（`concepts:`），将来在罗盘上悬停某一宿、某一卦时，能反查出所有讨论过它的段落。
