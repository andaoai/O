# 古籍笔记

本处按书立卷，按卷立章，按章立段。每一段落有独立编号（如 `yisizhan.01.001`），既方便自身回顾，也为将来罗盘与文本的**双向联动**留下锚点。

段落 frontmatter 中会声明它讨论了哪些**罗盘环**（`rings:`）与**概念**（`concepts:`），将来在罗盘上悬停某一宿、某一卦时，能反查出所有讨论过它的段落。

## 个人评估书籍标准

所选每一本书，本人心里都会先过一遍以下筛子。**书籍非圣典，皆是人手所记**——录入前须知其来处，读时才不至于把注家的观点当作原始事实。

### 需要警惕的七条

1. **政治正确问题** —— 历代星占官修书（如《开元占经》）往往为皇权服务，某些"应验"记录是事后附会。凡涉及天变对应人事之处，先看编撰背景，再论其观察本身。
2. **星辰偏移问题** —— [岁差](/concepts/precession) 使古今星宿位置约相差 20° / 千年。古书所记的宿度、节气星象，须以其著述年代回算，切勿直接套用当代天球。
3. **时代特色带来的语义修改** —— 同一个词，汉唐宋明的用法可能完全不同（如"消息"、"纳甲"、"游魂"），先秦典籍在汉代被重新诠释、宋人再度重构，语义层层叠加。读汉唐书要能剥离宋以后注疏。
4. **手抄与人伪** —— 传世本经历千百年抄刻，讹字、脱简、后人窜入、伪托古人诸家层出。《京氏易传》今传本与《汉书·京房传》所述已不完全一致；《易纬》八种真伪不一，须逐篇甄别。
5. **时间维度的沉淀** —— 一本书能穿越千年仍被后世大家一再引用、注疏、辨正，本身就说明它在时间洪流中被反复淘洗过。没有价值的书自会被遗忘，能被历代高手认可的，必有其天星福照护持的缘分。选书优先看"被谁读过、被谁引过"，不看"被谁转发过"。
6. **交流门槛的自守** —— 读一两遍就自认懂了、不做数据回算、不作实操验证的对话，本人不参与。若你尚未走完自己该走的路径，此处内容你会看不懂；若你的阅读量与实证经验能与我同频、乃至超过我，欢迎具体质疑、逐条辩驳。**泛化否定与情绪化站队，一律不予回应。**
7. **"理解"二字的分量** —— 所谓理解，"理" 是能把脉络理顺、条分缕析；"解" 是要用身体的五脏六腑去消化、要用时间去沉淀。**读懂 ≠ 理解**：仅在文字层面能复述，只是"读懂"；把它嚼进骨血、能在自己的语境里**重构并创新**——那才叫"理解"。本站笔记的最终考验，不是"我读了多少书"，而是"我能否以之构建出前人所未言之物"。

### 底层立场

- **一切书籍源于对自然天象的观察**。二十八宿、二十四节气、六十甲子、六十四卦，本原皆为**可观察、可复现、可记录的自然现象**——星辰位置、日月盈亏、气候变化、生物物候。非玄学概念。
- **要有逻辑可言，可通，且要论证**。凡遇到某说法，先问：它对应哪个天象？其推导链条是否闭合？如闭不上，指出断点，不强行圆说。
- **单本书籍论证 = 孤证**，仅作参考。同一命题若无多重来源交叉印证（不同时代、不同流派、乃至跨文明的希腊 / 印度 / 伊斯兰天文），暂列存疑。
- **数据是工具，不是世界本身**。数学、算法、天文回算都是**观察天象的工具**——是接近真实的路径，不是真实本身。任何模型都有它的近似阶数、有它无法涵盖的边界（岁差长期项、大气折射、章动短周期、地磁扰动……）。凡幻想"某个公式可以完美精准地描述世界"的，等价于让自己坐上上帝的位置——**上帝不会来当，你也不必装当**。工具能让我们看得更远、算得更细，但天地本身永远大于任何一次拟合。用数据校验古书是好事，但数据算出来的结果**只是当前工具能给的最佳近似**，不是终审判决。
- **算法的第一性 = 加法**。无论多复杂的推步、多花哨的历算，剥到最底层都是**从一个已知位置累加一个基本步长**——干支纪日是加 1、二十四气是加 15° 黄经、六爻纳甲是加固定爻位间距、开普勒方程本质是迭代加法。加法之所以是万法之首，因为它对应"生发"这一原动作：从无到有加出第一步，才有后续的减、乘、除、幂、卷积。汉字"甲"从音（jiǎ / gā）到义（"甲为万物之始"，《说文》"甲，东方之孟，阳气萌动"），都在讲同一件事——**首位、萌发、加的起点**。算法看到底，是在跟"甲"打交道。
- **人是宇宙的投影，不是宇宙的中心**。追到"人的底层"，一定会追到天上：日月五星（七政）供给的是**身**——阴阳寒暑、气血节律、五脏应五行；二十八宿供给的是**神**——方位、时令、乃至思维的坐标系。故《黄帝内经·灵枢·邪客》曰"天有日月，人有两目；天有列星，人有牙齿"——这不是比喻，是**同构映射**：天体的秩序是原图，人的形神是投影。数学里最迷人的概念之一就是**投影**：一个高维体（球、多胞形、流形）在低维平面上留下影子，影子看着是"新东西"，其实只是原体在某个方向上的截面。**我并非我，我是宇宙这个高维体在某一时空点、某一方位角上的投影**——生辰八字之所以能读，命宫星度之所以能推，就是因为投影方向不同、影子形态自然不同。一切术数底层皆为投影几何：黄道→赤道是投影、天球→平面是投影、天象→人身也是投影。想读懂古书，先接受这个观法。

- **个人学习为主**。本人时间有限，无学术团队与专业校勘条件，所有结论均为一人之见，欢迎具体质疑，不接受泛化否定。
