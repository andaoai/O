# 京氏易传

> 西汉 · 京房 撰 · 约成书于汉元帝建昭年间（公元前48–前33）

## 书籍导读

《京氏易传》三卷，是**现存最早、最完整的象数易学著作**，也是后世纳甲占卜（六爻预测）的理论源头。

作者京房（公元前77–前37），东郡顿丘（今河南清丰）人，西汉著名易学家。本姓李，因其爱好音律，擅吹律定姓，故改姓京。师从焦延寿（焦赣），深得《易》学真传，后因劾奏石显等权臣，被下狱处死，年仅四十。

《京氏易传》的核心贡献在于：将**六十四卦重组为八宫体系**，为每一卦注入**五行、天干地支、世应飞伏**等参数，使《周易》从一部哲学经典转化为**可操作、可推演、可占卜的动态系统**。

书名"易传"不同于《十翼》中的《易传》，京房所传乃其师焦氏之学，自成一家。其理论被后世纳甲筮法（火珠林法）全面继承，至今仍是民间占卜的主流方法之一。

::: warning 编者按
京房本名李房，改姓京氏，故后世或称"京君明"（君明为其字）。其易学上承焦赣，下启虞翻、干宝诸家，在汉易中独树一帜。今传本《京氏易传》经宋代以来多次校订，卷次与篇目或有调整，请见 [卷上 · 乾卦](/books/京氏易傳/乾卦)。
:::
