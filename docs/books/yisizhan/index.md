# 乙巳占

> 唐 · 李淳风 撰 · 约成书于唐高宗麟德二年（665）

## 书籍导读

《乙巳占》十卷，是**现存最完整的唐代天文占候类著作**，也是了解隋唐时期天文历法与占验体系的第一手材料。

作者李淳风（602–670），岐州雍县人，唐初著名天文学家、数学家，曾任太史令，主持修订《戊寅元历》为《麟德历》，亦是《晋书·天文志》《隋书·天文志》的主要作者之一。

书名"乙巳"取自作者出生之年——**隋仁寿二年（602）壬戌**……

::: warning 编者按
上面这句是**故意留错**的用例——李淳风生于壬戌而非乙巳。这类校勘记录会写在段落的「校注」部分。
"乙巳"更可能取自书成前后的某个乙巳纪年（如唐太宗贞观十九年 645 或高宗麟德二年前后），此处待考，请见 [卷一·天说 · 疑问部分](/books/yisizhan/juan-1/tianshuo#疑问)。
:::
