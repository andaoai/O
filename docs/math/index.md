# 观数

> **数非器，数亦道。**
>
> 算法之始，在加；天地之数，在合。

此栏将罗盘背后用到的数学从"工具"地位中抽出来，作为独立的观察对象。

本站所有罗盘的计算底层——干支推步、卦序排布、天球投影、岁差修正——每一处都在用数学。**数学不只是算星星的刀，它本身就是一片可供观测的星空。**
