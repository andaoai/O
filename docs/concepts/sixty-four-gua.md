# 六十四卦 · 序卦

**六十四卦**由八经卦两两相重而成——上卦、下卦各三爻，合六爻定一卦。六十四卦的排列有多种「序」：

- **通行本**：《周易》上下经，乾坤起，未济终，义理次第；
- **先天序**（伏羲 / 邵雍）：按二进制位反转排布，乾南坤北，四正卦定位，构成完美对称的圆图；
- **京房八宫**：以八纯卦为宫首，每宫领七变卦（[八宫卦](./eight-palace)）。
