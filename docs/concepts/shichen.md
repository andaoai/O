# 十二辰 · 十二时辰

**十二地支**（子丑寅卯辰巳午未申酉戌亥）既用于纪日、纪年，也用于将一日等分十二段，每段两小时，是为「时辰」——子时（23-1）夜半、午时（11-13）日中……

十二辰另有一层空间含义：作为方位坐标——**子北、午南、卯东、酉西**，与二十八宿、四象、八卦共同构成传统的天圆分区。
