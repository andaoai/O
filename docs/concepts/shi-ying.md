# 世应爻位

在京房八宫卦系里，每卦的六爻中固定有两爻标注：

- **世爻**：变化起点，占卜中代表「己方 / 主体」。世爻的位置对应该卦在八宫中的变化次第——
   - 一世卦初爻为世，二世卦二爻，……上五世；
   - 游魂卦四爻为世，归魂卦三爻为世；
   - 八纯卦上爻（六爻）为世。
- **应爻**：与世爻相隔三爻的对应位。若世在初爻则应在四爻，二对五，三对六——世应间隔恒为三爻。

占卜时通常关注世应两爻的动变、五行生克，作为吉凶判断的骨干。下方外环为八宫六十四卦，内环仅显示每卦的世 / 应位标记：
