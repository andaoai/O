# 二十四节气 · 节气与中气

**二十四节气**是太阳沿黄道每移 15° 一分的时刻标记。奇数位者称「节」（立春、惊蛰、清明……），偶数位者称「中气」（雨水、春分、谷雨……）。这一「节 / 中气」二分法是古代阴阳合历置闰的关键：

> **凡朔在中气之前者，无中气之月即为闰月。**

节气与中气在下环以不同颜色标注，随当前时间自动高亮邻近节气：
