# 七曜 · 日月五星

**七曜**：日、月、金、木、水、火、土——古人所观察到的沿黄道运行、位置日日不同的七颗「动星」。七曜是古代历法、术数、占验共同的骨架：

- **日、月** 定阴阳、明晦、朔望；
- **五星** 与五行相配：木星（岁星）· 火星（荧惑）· 土星（镇星）· 金星（太白）· 水星（辰星），各主一方一气；
- 七日一周（曜日）源自七曜——星期一至星期日的西源同此系统。

五星在黄道上并非匀速：有**顺行**（黄经增大）、**留守**（近乎不动）、**逆行**（黄经反减）三态。七曜环把这三态用高亮层级与径向刻线可视化，同时标注每颗星当下所入的**宿**及**宿内度数**：
