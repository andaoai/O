# 干支 · 六十甲子

**天干十位**（甲乙丙丁戊己庚辛壬癸）与**地支十二位**（子丑寅卯辰巳午未申酉戌亥）依次相配，六十为一周，即「六十甲子」。汉以后一贯用于纪年、纪月、纪日、纪时——「六柱」记时体系的基础字符集。

- **天干配五行**：甲乙木、丙丁火、戊己土、庚辛金、壬癸水；
- **地支配五行**：寅卯木、巳午火、辰戌丑未土、申酉金、亥子水；
- **纳音**：六十甲子每两柱一组配一种五行「纳音」，另成一套象数体系（见 [六十甲子六环](/compass/liushi-jiazi)）。

下环从外到内为「六十甲子」「十二地支」「十天干」三层叠合。当日日柱按五行自动配色高亮。
