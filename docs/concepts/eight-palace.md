# 八宫卦

**京房八宫**是六十四卦的另一种排列：以八纯卦（乾兑离震巽坎艮坤）各为一「宫」首，每宫依次演出「一世 → 二世 → 三世 → 四世 → 五世 → 游魂 → 归魂」七变，共 8 × 8 = 64 卦。

- 变化的顺序由**世爻**从初爻起向上推移决定；
- **游魂**回到四爻反转，**归魂**下卦复原；
- 每卦配定**世 · 应爻**（间隔三爻的呼应关系）——占卜时以世为己、应为他。
