# 四象 · 青龙白虎朱雀玄武

四象是二十八宿的顶层分组——四方各七宿合为一「灵物」：

- **东方 · 青龙**：角亢氐房心尾箕；
- **南方 · 朱雀**：井鬼柳星张翼轸；
- **西方 · 白虎**：奎娄胃昴毕觜参；
- **北方 · 玄武**：斗牛女虚危室壁。

四象既是**空间方位**（东南西北），也是**季节**（春夏秋冬）与**五行**（木火金水）的象征体系，中央加土（皇极）即成五方。
