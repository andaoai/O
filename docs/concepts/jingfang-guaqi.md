# 京房卦气 · 六日七分

西汉京房（前 77–前 37）以《周易》配一年时序：**六十四卦匀分一年**，每卦当值约 6 又 7/80 天，即「六日七分」（365.25 ÷ 60 ≈ 6.0875）。冬至日为「中孚」卦起，依先天次序（复卦 · 冬至 · 一阳生 → 乾卦 · 立夏前 · 六阳极 → …）行进。

四正卦（震春、离夏、兑秋、坎冬）分司四立，十二辟卦（消息卦）分司十二月，其余分佈六十日下。

下环以**冬至为 0°**排布——外圈六日七分刻度，内圈六十卦（辟卦 / 侯卦 / 大夫卦分色）：
