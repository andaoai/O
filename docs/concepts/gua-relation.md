# 飞伏 · 显隐互藏

**飞伏**是京房八宫体系的核心概念。「飞」为显现之卦（当前所见之卦），「伏」为隐藏之卦（飞卦背后所藏的对应之卦）。阴阳互藏其宅——飞显则伏隐，飞隐则伏显。

## 规则

飞伏的取法因卦的世位类型而异：

| 世位类型 | 伏卦规则 | 举例 |
|----------|---------|------|
| 本宫（八纯卦） | 飞伏互取对宫：63 - feiValue | 乾→坤，坤→乾 |
| 一世 · 二世 · 三世 | 伏 = 新内卦（下卦）之疊卦 | 姤（内巽）→巽 |
| 四世 · 五世 | 伏 = 新外卦（上卦）之疊卦 | 观（外巽）→巽 |
| 游魂 | 伏 = **五世卦**之外卦的疊卦 | 晋（剥之外卦艮）→艮 |
| 归魂 | 伏 = **游魂卦**之内卦的疊卦 | 大有（晋之内卦坤）→坤 |

> **疊卦**：三爻经卦重叠为六爻。如巽（☴）疊→䷸ 巽为风（value=54）。

## 示例：飞伏读法

### 乾卦（本宫卦）

```
飞：䷀ 乾  →  伏：䷁ 坤
```

乾为纯阳之象，其伏为纯阴之坤。显者为天，隐者为地——阳中藏阴，阳极阴生。

### 姤卦（一世卦）

```
飞：䷫ 姤  →  伏：䷸ 巽
```

姤卦内卦由乾变巽（一阴生），其伏取本宫内卦的原卦——乾坤未变时内卦本为乾，但这里是另一种读法：新内卦自为飞，其伏为**新内卦本身的叠卦**。姤之内卦为巽，故伏为巽。
