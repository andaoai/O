# 二十八宿 · 距星

沿赤道 / 黄道附近选定 28 组恒星群，谓之「二十八宿」（也作二十八舍、二十八星）。每宿以其中一颗最亮或特征最明显的星为「距星」，宿的赤经宽度由此星到下一宿距星测得。二十八宿是古代天官体系的骨架。

四方各七宿：

- **东方苍龙**：角、亢、氐、房、心、尾、箕；
- **北方玄武**：斗、牛、女、虚、危、室、壁；
- **西方白虎**：奎、娄、胃、昴、毕、觜、参；
- **南方朱雀**：井、鬼、柳、星、张、翼、轸。
